# Maths Guide — Presentation Template 02

A Replit-based interactive presentation template for GCSE Maths lessons. Fork and remix for your own curriculum content.

## Tech Stack

- TypeScript / Node.js
- React + Vite
- Drizzle ORM
- Tailwind CSS

## Status

🟡 Legacy — template reference only. Not actively maintained.

## Usage

1. Fork on Replit
2. Edit curriculum content in `client/`
3. Deploy via Replit
